# 🎮 寶可夢注音符號練習表

一個用於生成寶可夢注音符號練習表的 Python 專案，讓小朋友可以看著寶可夢圖片和中文名稱來練習填寫注音符號。

**其他語言 / Other Languages:** [English](docs/i18n/README_en.md) | [日本語](docs/i18n/README_ja.md)

## 🌟 專案特色

- 📸 **完整資料庫**：包含 1025 隻寶可夢圖片與中文名稱
- 🎯 **世代選擇**：支援單一世代或多世代組合生成
- 📄 **A4 優化**：橫式列印版面，適合實際使用
- ✨ **客製化設計**：支援字型、字級、邊框等多項調整
- 🚀 **快速載入**：智能分頁設計，小檔案快速開啟
- 🖊️ **無邊框設計**：極簡風格，專注內容填寫

## 📋 功能概覽

### 核心功能
- 🎮 **寶可夢資料收集**：自動下載官方圖片與中文名稱
- 📝 **練習表生成**：互動式世代選擇器
- 🛠️ **統合管理工具**：一鍵完整設置與資料管理

### 📝 使用小貼士
- 🌐 **線上預覽**：點擊「🌐 線上預覽」可直接在瀏覽器查看練習表效果
- 📥 **下載檔案**：點擊「📥 下載」保存 HTML 檔案到電腦，可離線使用
- 📄 **轉存 PDF**：詳細步驟請參考下方「PDF 轉存教學」
- 🖨️ **列印設定**：建議使用 A4 橫式列印，效果最佳
- ✏️ **練習方式**：讓小朋友看著寶可夢圖片和名稱，在空白欄位填寫注音符號
- 🎯 **專用字型**：內建寶可夢注音專用字型，清晰易讀
- 📚 **分批使用**：可以先從第一世代開始，循序漸進

### 📄 PDF 轉存教學

🎯 **將練習表轉存為 PDF 檔案，方便保存和分享**

#### 📋 轉存步驟（適用所有瀏覽器）

1. **🌐 開啟線上預覽**
   - 點擊任一世代的「🌐 線上預覽」連結
   - 等待頁面完全載入（確保所有寶可夢圖片都顯示）

2. **🖨️ 開啟列印功能**
   - **Windows**：按 `Ctrl + P` 
   - **Mac**：按 `⌘ + P`
   - **或右鍵選單**：右鍵點擊頁面 → 選擇「列印」

3. **⚙️ 設定列印選項**
   ```
   📄 目的地：儲存為 PDF / Save as PDF
   📐 紙張大小：A4
   📏 方向：橫向 (Landscape)
   🖼️ 選項：✅ 背景圖形 (Background graphics)
   📊 邊界：最小邊界 (Minimum margins)
   🎨 色彩：彩色 (Color)
   ```

4. **💾 儲存 PDF**
   - 點擊「儲存」或「Save」
   - 選擇儲存位置
   - 檔案名稱建議：`寶可夢第X世代注音練習表.pdf`

#### 🔧 各瀏覽器特殊說明

- **🌐 Chrome / Edge**：列印選項中直接選擇「儲存為 PDF」
- **🦊 Firefox**：目的地選擇「Microsoft Print to PDF」或「儲存為 PDF」  
- **🍎 Safari**：點擊左下角「PDF」選單 → 選擇「儲存為 PDF」

#### 💡 PDF 品質優化小技巧

- ✅ **載入完成**：確保所有寶可夢圖片都完整顯示後再列印
- ✅ **背景圖形**：務必勾選「背景圖形」選項，保留邊框和底色
- ✅ **橫向列印**：選擇橫向可確保表格不被截斷
- ✅ **邊界設定**：使用最小邊界可容納更多內容

## 🚀 快速開始

### 環境需求
- Python 3.8+
- 網路連線 (用於下載圖片)
- 約 150MB 磁碟空間

### 安裝步驟
```bash
# 1. 複製專案
git clone https://github.com/tk009999/pokemon-phonetic-practice.git
cd pokemon-phonetic-practice

# 2. 建立虛擬環境
python3 -m venv venv
source venv/bin/activate  # macOS/Linux
# 或 venv\Scripts\activate  # Windows

# 3. 安裝依賴
pip install -r requirements.txt

# 4. 執行統合管理工具
python pokemon_manager.py
```

### 快速生成練習表
```bash
# 直接生成特定世代
python generate_generation_selector.py
```

## 🛠️ 使用說明

### 世代選擇器
執行 `generate_generation_selector.py` 快速生成練習表：

- 支援單一世代：`1` (第一世代)
- 支援多世代：`1,2,3` (前三世代)
- 支援範圍：`1-3` (第一到第三世代)
- 支援全部：`10` (全部九個世代)
